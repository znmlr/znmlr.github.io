# 简介

cmder是一个跨平台的命令行增强工具，可以集成windows batch、power shell、git、linux bash等多种命令行于一体，十分方便 

  <!--more-->

# 安装

## 下载

[github地址](https://github.com/cmderdev/cmder/releases/)

[官网](http://cmder.net/)

## 安装

随意解压到任意目录，然后添加到``环境变量``下

## 右键快捷打开目录

管理员权限打开cmder，执行下面的命令

```
cdmer /register user
// 或
cmder /register all
```
